Introduccion Solucion



Durante la construcción de este proyecto se busca innovar una solución tecnológica en los cuartos de soluciones la cual facilite el control de las entradas, salidas y ubicación de las mercancías, teniendo en cuenta la capacidad disponible que el mismo tenga.


Para darle solución y que se pueda integrar con el sistema Core de negocio vamos a utilizar un api que se conecte con el servidor y podamos darle solución al objetivo del proyecto



Objetivos


  • Administrar de manera eficiente la mercancía ingresada a los cuartos de soluciones de la empresa y poder visualizar las diferentes mercancías en sus respectivos lugares, Suministrar un lugar específico a la mercancía ingresada y a la vez facilitar al usuario la información necesaria.


  • Optimizar el inventario de las mercancías ingresadas a los distintos cuartos de soluciones y así poder lograr darle rotación a la mercancía dando solución a la necesidad que en este caso serían los espacios disponibles en las zonas.



Tecnologias Utilizadas Para el Desarrollo :


  • Angular

    Para el desarrollo de este sitio web,utilizamos este Framework el cual nos facilito la creacion del front end con la cual logramos el objetivo de un sitio web facil y util de usar para los usuarios;ademas de cumplir con el reto propuesto por la compañia


  • Spring Boot

    Para el funcionamiento de este sitio web utilizamos este framework para crear una api la cual se conecta con la base de datos y nos permitio hacer las funciones de este sitio web

  • Sweet Alert

    Con este framework nos facilita los mensajes de entrada y los mensajes de verificacion y confirmacion

  • My SQL

    Con este editor de SQL creamos la base de datos de nuestro proyecto final
